# 🏭 MES 互动式学习平台

## 📖 项目简介

基于高端制造业MES（制造执行系统）的操作指南和功能清单，构建的**互动式学习平台**。通过知识学习、模拟操作和功能探索三大模块，帮助用户全面掌握 MES的核心概念与操作技能。

> 🎓 零依赖、纯前端，打开浏览器即可开始学习！

---

## 🎯 核心功能

### 模块1: 📚 知识学习与测验

- **7 大知识模块**：MES概述、计划与工单、批次管理、设备管理、质量管理、物流与库存、高级功能
- **32 个知识点**：涵盖 MES 系统的核心概念、架构、操作流程
- **62 道测验题目**：每个知识点配套主题题 + 补强题
- **自适应出题系统**：
  - ✅ 答对主题 → 进入下一知识点
  - ❌ 答错主题 → 出补强题
  - ✅ 补强题答对 → 标记为「补强后通过」
  - ❌ 补强题答错 → 标记为「需复习」

### 模块2: 🔄 批次模拟器

- **交互式 Lot 生命周期状态机**：SVG 内联绘制可视化状态图
- **8 种状态**：已创建、已开始、等待中、运行中、已扣留、已完成、已出货、已报废
- **22 种操作**：覆盖完整生产场景，包括 TrackIn/TrackOut、Hold/Release、Split/Merge、Rework/Branch 等
- **实时操作日志**：记录每一步操作的时间和状态变化
- **示例工序流程**：8 道工序模拟真实生产线

### 模块3: 🗺️ 功能地图

- **8 大分类、129 个功能项**：完整展示 MES 系统功能清单
- **可展开树形结构**：层级化浏览，清晰直观
- **搜索过滤**：快速定位目标功能
- **中英文对照**：每个功能项包含英文名称和详细描述

### ✏️ 自适应测验

测验系统采用自适应出题策略，根据答题表现动态调整：

```
答对主题题 ──→ ✅ 通过 ──→ 进入下一知识点
    │
答错主题题 ──→ 出补强题
                  │
            答对补强题 ──→ ⚠️ 补强后通过
                  │
            答错补强题 ──→ ❌ 需复习
```

### 🔄 批次模拟器

8 种状态覆盖 Lot 完整生命周期：

```
已创建 ──→ 已开始 ──→ 等待中 ←──→ 运行中
                        │              │
                        ├──→ 已扣留 ←──┘
                        │       │
                        │       ├──→ 已报废
                        │       │
                        ↓       │
                     已完成 ←───┘
                        │
                        ↓
                     已出货
```

22 种操作覆盖完整生产场景：

| 分类 | 操作 |
|------|------|
| 基础 | 开始批次、进入工序 |
| 跟踪 | 进站 TrackIn、出站 TrackOut、取消进站 |
| Hold | 扣留批次、运行中扣留、释放扣留 |
| 报废 | 报废批次、反报废 |
| 批次操作 | 拆批 Split、合批 Merge |
| 工序操作 | 变更工序、返回工序、跳过工序 |
| 高级 | 返工 Rework、完成返工、分支 Branch、结束分支 |
| 完成 | 完成批次 |
| 出货 | 出货 Ship、取消出货 |
| Bank | 入Bank、出Bank |
