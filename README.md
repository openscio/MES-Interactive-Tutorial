# 🏭 MES 互动式学习平台

> **当前版本：v1.1** | 🎓 零依赖、纯前端，打开浏览器即可开始学习！

## 📖 项目简介

基于高端半导体制造业 MES（制造执行系统）的操作指南和功能清单，构建的**互动式学习平台**。通过知识学习、自适应测验、批次模拟（含 20 个引导式真实生产场景）和功能地图四大模块，帮助用户全面掌握 MES 的核心概念与操作技能。

---

## 🎯 核心功能

### 模块 1：📚 知识学习与测验

- **7 大知识模块**：MES 概述、计划与工单、批次管理、设备管理、质量管理、物流与库存、高级功能
- **32 个知识点**：涵盖 MES 系统的核心概念、架构、操作流程
- **62 道测验题目**：每个知识点配套主题题 + 补强题
- **自适应出题系统**：
  - ✅ 答对主题 → 进入下一知识点
  - ❌ 答错主题 → 出补强题
  - ✅ 补强题答对 → 标记为「补强后通过」
  - ❌ 补强题答错 → 标记为「需复习」

### 模块 2：🔄 批次模拟器（双模式）

#### 自由模式（Free Mode）

- **交互式 Lot 生命周期状态机**：SVG 内联绘制可视化状态图
- **8 种状态**：已创建、已开始、等待中、运行中、已扣留、已完成、已出货、已报废
- **22 种操作**：覆盖完整生产场景，包括 TrackIn/TrackOut、Hold/Release、Split/Merge、Rework/Branch 等
- **实时操作日志**：记录每一步操作的时间和状态变化
- **示例工序流程**：8 道工序模拟真实生产线

#### 场景模式（Scenario Mode）🆕

**6 大类、20 个引导式真实生产场景，共 64 个操作步骤**，涵盖半导体制造中常见的特殊处理场景：

| 类别 | 图标 | 场景数 | 包含场景 |
|------|:----:|:------:|----------|
| **Lot Hold（批次暂停）** | ⏸️ | 4 | 质量异常 Hold、设备关联 Hold、量测异常 Hold、客户变更 Hold |
| **Lot Scrap（报废）** | 🗑️ | 3 | 整批报废、部分报废（拆批+报废）、碎片处理 |
| **Lot Rerun/Reclaim（重工）** | 🔁 | 3 | 光刻返工、薄膜去除重长、晶圆回收 |
| **Priority Change（优先级）** | 🚀 | 4 | Hot Lot 加急、Super Hot Lot 特急、优先级提升、合批运行 |
| **Special Handling（特殊处置）** | 🔬 | 3 | 工程试验批次、挡控片 / Monitor Wafer、客户特殊要求 |
| **Fab External（厂外因素）** | ⚡ | 3 | 缺料 / 缺 Mask、停电 / 断气、系统锁死 |

**场景模式特色：**

- 📖 **叙事背景**：每个场景附带真实的产线背景故事和学习目标
- 🎯 **步骤引导**：逐步指引正确操作，期望按钮脉冲高亮提示
- 💡 **渐进提示**：操作错误时提供提示，2 次错误后自动展示 Hint
- ✅ **操作验证**：实时校验每步操作是否正确，给出成功/错误反馈
- 📊 **完成总结**：完成后显示关键要点和实际工厂注释
- 🏆 **进度追踪**：通过 localStorage 记录已完成场景，浏览器显示完成徽章
- 🏷️ **难度分级**：入门（9）、中级（9）、高级（2）三级难度标签

### 模块 3：🗺️ 功能地图

- **8 大分类、129 个功能项**：完整展示 MES 系统功能清单
- **可展开树形结构**：层级化浏览，清晰直观
- **搜索过滤**：快速定位目标功能
- **中英文对照**：每个功能项包含英文名称和详细描述

### 🌙 深色模式

- 一键切换浅色 / 深色主题
- 基于 CSS 变量的全局主题系统，覆盖所有模块和组件
- 主题偏好通过 localStorage 持久化保存

### ✏️ 自适应测验

测验系统采用自适应出题策略，根据答题表现动态调整：

```
答对主题题 ──→ ✅ 通过 ──→ 进入下一知识点
    │
答错主题题 ──→ 出补强题
                  │
            答对补强题 ──→ ⚠️ 补强后通过
                  │
            答错补强题 ──→ ❌ 需复习
```

### 🔄 批次模拟器

8 种状态覆盖 Lot 完整生命周期：

```
已创建 ──→ 已开始 ──→ 等待中 ←──→ 运行中
                        │              │
                        ├──→ 已扣留 ←──┘
                        │       │
                        │       ├──→ 已报废
                        │       │
                        ↓       │
                     已完成 ←───┘
                        │
                        ↓
                     已出货
```

22 种操作覆盖完整生产场景：

| 分类 | 操作 |
|------|------|
| 基础 | 开始批次、进入工序 |
| 跟踪 | 进站 TrackIn、出站 TrackOut、取消进站 |
| Hold | 扣留批次、运行中扣留、释放扣留 |
| 报废 | 报废批次、反报废 |
| 批次操作 | 拆批 Split、合批 Merge |
| 工序操作 | 变更工序、返回工序、跳过工序 |
| 高级 | 返工 Rework、完成返工、分支 Branch、结束分支 |
| 完成 | 完成批次 |
| 出货 | 出货 Ship、取消出货 |
| Bank | 入 Bank、出 Bank |
